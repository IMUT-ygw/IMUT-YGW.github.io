一、配置metricbeat

四、结论

根据日志信息可知：存在大部分调研需求的内容，少部分内容名称不精确属于未知状态但有相似字段，需要时根据字段可进一步获取。

名称 是否存在 内容属性 补充
集群名称 是 调研需求 可获取到但本地为搭建集群
topic名称 是 调研需求 无
分区名称 是 调研需求 无
采集开始时间 未知 调研需求 存在采集开始时间戳
采集结束时间 未知 调研需求 未发现结束标志
current_offset 未知 调研需求 存在partition.offset.newest
log_end_offset 未知 调研需求 存在partition.offset.oldest
lag 是 调研需求 存在consumergroup.consumer_lag
消费者组 是 调研需求 无
监控侧会根据topic名称 是 调研需求 无
分区名称区分业务 是 调研需求 无

注意：此次调研es、metricbeat、kibana版本为7.6.0，kafka版本为2.11-2.2.2，zookeeper版本3.5.8，不同版本之间可能存在不同，以实际情况为准。
